== Description ==

Plugin sends Contact Form 7 (first name, last name, email, phone) to HubSpot CRM contact.

Installation:

* Go to "Credentials to API HubSpot" tab, enter API key, press save.
* Go to "Company list" tab, copy CF7 short code.
* Go to CF7 form editor, paste it at the bottom form.
* Send your test entry to HubSpot.
* Plugin Setup page: "Settings" - "HubSpot"

Map contact form 7 fields:

* Contact form 7 must have field: [usr_email]. Optional fields: [usr_fname] [usr_lname] [usr_phone].
* Contact form 7 short code, with Audience ID [hidden wm_addcontact_compid default:"1234567"]
